# Dialogue Platform

A web application that enables registered users to create discussion topics and exchange opinions.

## Features

* **User Authentication:** Registration, login, and session management.
* **Password Management:** Functionality for users to update their credentials.
* **Topic Creation:** Allows users to start new discussion threads.
* **Message Posting:** Allows users to submit posts under existing topics.
* **Asynchronous Communication:** Dynamic updates using JSON and the Fetch API.
* **Client-side Validation:** Form verification using JavaScript.

## Tech Stack

* **Backend:** Java Servlets, JSP, JDBC
* **Database:** MySQL
* **Frontend:** HTML, CSS, JavaScript
